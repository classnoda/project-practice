## ✨ Esp32
Проект написан на **MicroPython v1.27** и предназначен для микроконтроллера **ESP32**. Данный код имеет свои преимущества:
- **BLE UART-протокол** — управление осуществляется через стандартный Nordic UART Service (NUS), что позволяет использовать любое приложение с поддержкой BLE UART (например, *nRF Connect*, *Serial Bluetooth Terminal*).
- **Чистая модульная архитектура** — логика разделена на независимые модули: [`BLE_Controller`](esp32/modules/BLE_Controller.py) и [`Motor`](esp32/modules/Motor.py), что упрощает поддержку и расширение проекта.
- **Централизованная конфигурация** — все пины и параметры вынесены в [`config.py`](esp32/config.py), смена железа требует правки только одного файла.
- **Высокая частота ШИМ** — моторы управляются ШИМ на частоте 50 кГц, что обеспечивает плавный ход и снижает акустический шум.
- **Оптимизация под MicroPython** — критические методы помечены декораторами [`@micropython.native`](modules/Motor.py:23) и [`@micropython.viper`](modules/Motor.py:61) для максимальной скорости выполнения без перехода на C.
- **Режим торможения и сна** — реализованы методы `brake()` (жёсткое торможение) и `sleep()` / `wake_up()` (управление линией STBY драйвера) для экономии энергии.
- **Поддержка коррекции направления** — флаги `offsetA` / `offsetB` в конфиге позволяют менять направление вращения каждого мотора без перепайки.

## 🎮 Управление

Команды отправляются по BLE как одиночные байты:

| Клавиша | Действие         |
|---------|-----------------|
| `w`     | Вперёд          |
| `s`     | Назад           |
| `a`     | Влево           |
| `d`     | Вправо          |
| `e`     | Плавная остановка |
| `r`     | Торможение (brake) |
